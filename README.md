# Modern Developer Portfolio (2026 Edition)

A high-performance, accessible, and stunning developer portfolio built with the latest 2026 tech stack.

## Tech Stack

- **Next.js 15** (App Router, Server Components)
- **React 19**
- **TypeScript**
- **Tailwind CSS v4**
- **Framer Motion** for animations
- **Lucide React** for icons
- **Dark/Light Mode** support with `next-themes`
- **Responsive Design** (Mobile-first)

## Features

- **Hero Section**: Personal branding and CTA for resume/projects.
- **About Section**: Detailed professional bio.
- **Tech Stack**: Categorized skills with interactive icons.
- **Projects**: Showcase of work with tech badges and demo links.
- **Experience**: Timeline of professional and educational milestones.
- **Contact**: Easy access to social profiles and contact info.

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Assets

- **Profile Photo**: `public/myimage.png`
- **Resume**: `public/resume.pdf`
- **Project Screenshots**: `public/projects/[slug]/desktop.png` and `mobile.png`
